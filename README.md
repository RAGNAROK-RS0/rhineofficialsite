## Quick Start

### Frontend (HTML/CSS/JS)
```bash
cd frontend
# Open index.html in browser
```

### PHP Webshop
```bash
cd php
php -S localhost:8000
# Open http://localhost:8000
```

### Python Demos
```bash
cd python
python chatbot.py
python bookmanager.py
```
